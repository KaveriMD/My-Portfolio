Single Page Application for Social Feeds
Developed a Single Page Application (SPA) for a social feed listing screen using React.js, designed to simulate basic social media interactions. The application allows users to post comments dynamically, which are instantly rendered on the feed without requiring a page reload. To enhance user experience and maintain state persistence across sessions, local storage was utilized to store and manage user comments. Users can also delete comments, with changes immediately reflected in both the UI and the stored data. This project demonstrates practical experience in building interactive UIs, managing component states, handling user events, and working with browser storage APIs in a React environment.
Calendar Appointment App
Designed and developed a dynamic calendar-based appointment management UI using React.js, enabling users to view all appointments for a specific day and seamlessly navigate between different days. The application supports full CRUD (Create, Read, Update, Delete) operations for appointments. It features conflict avoidance logic that ensures no overlapping appointments are created. On the backend, built a robust RESTful API using C# and .NET to handle appointment data, with proper validation and error handling. Thorough unit testing and integration testing were conducted to ensure backend reliability and to eliminate bugs, ensuring consistent performance and data accuracy. This project highlights strong skills in full-stack development, user experience design, API creation, and test-driven development.
